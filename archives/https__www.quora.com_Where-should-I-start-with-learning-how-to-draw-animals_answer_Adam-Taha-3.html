If you want to draw animals then you need to have already a basic understanding of form, shapes.

You’d know everything is about form, shape..but also, they have a function. Function meaning the pose of someone or animal, or an object has to be that way due to it’s structure, anatomy and how it works…

Get good at drawing just the shapes of animals but also get some basic understanding of the function of their body. It doesn’t need a lot of knowledge, just basic. Watch movies, see references, books observe nature documentaries which I do a lot of...

Now, when you put these together you’re not sketching blindly…
